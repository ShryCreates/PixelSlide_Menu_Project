# PixelSlide_Menu_Project
This project is a responsive sidebar navigation interface built using HTML and CSS. It features a sliding menu controlled by a checkbox toggle without JavaScript, along with smooth transitions, hover effects, custom fonts, Font Awesome icons, and a full-screen background layout for a modern and interactive user interface.

A responsive and modern sidebar navigation interface built using pure HTML and CSS. This project demonstrates how to create a sliding sidebar menu using a checkbox toggle technique — without JavaScript — along with smooth animations, hover effects, and a clean UI design.

⚙️ How It Works

The sidebar menu is controlled using an HTML checkbox input. When the checkbox is checked, CSS selectors trigger transitions that slide the sidebar into view and toggle the menu icons — creating an interactive effect without JavaScript.
